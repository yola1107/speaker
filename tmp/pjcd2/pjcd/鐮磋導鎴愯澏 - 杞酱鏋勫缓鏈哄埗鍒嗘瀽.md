## 一、轮轴构建概述

### 1.1 什么是轮轴（Reel）？

轮轴是老虎机游戏中用于生成游戏盘面的核心数据结构。

```
轮轴 = 一个长的符号序列（通常100个位置）

游戏盘面 = 从轮轴中截取的可见区域（3行×5列）

    轮轴（100个位置）
    ┌─────────────────────────────────────┐
    │ 1 2 3 1 2 5 6 7 3 4 1 2 3 ... 6 7 5 │
    └─────────────────────────────────────┘
              ↓ 截取3个连续位置
    ┌───┐
    │ 5 │  ← 可见区域第1行
    │ 6 │  ← 可见区域第2行（中心行）
    │ 7 │  ← 可见区域第3行
    └───┘
```

### 1.2 为什么需要轮轴？

| 目的 | 说明 |
|------|------|
| **控制概率** | 通过符号在轮轴中的分布控制出现概率 |
| **保证公平** | 预先确定的序列，非实时随机 |
| **可验证性** | 可以预先计算 RTP，确保合规 |
| **特性控制** | 控制特定符号的出现位置和频率 |

### 1.3 本游戏的轮轴特点

| 特点 | 基础模式 | 免费模式 |
|------|---------|---------|
| **数量** | 5 条（每列 1 条） | 5 条（每列 1 条） |
| **长度** | 100 个位置 | 100 个位置 |
| **生成时机** | 每 10 次 spin 生成一次 | 免费游戏开始前生成一次 |
| **使用方式** | 10 次 spin 内复用 | 整个免费模式期间复用 |

### 2.4 为什么 WILD 只在中间三列？

```
原因分析：

1. 游戏平衡
   - WILD 可以替代任何符号形成中奖
   - 如果第 1 列有 WILD，容易形成大量从左到右的中奖
   - 限制在中间列可以控制 WILD 的"威力"

2. 视觉效果
   - 中间列出现 WILD 更有戏剧性
   - 第 1 列中奖是"起点"，中间列 WILD 是"惊喜"

3. 数学模型
   - LineGame 从左到右判奖
   - 第 1 列符号决定中奖符号类型
   - 中间列 WILD 帮助完成连线

4. SCATTER 不限位置
   - SCATTER 用于触发免费游戏
   - 需要分布在所有列，增加触发可能性
   - 所以 SCATTER 替换应用于所有 5 条轮轴
```

### 2.6 Go 代码实现

```go
// ReelBuilder 轮轴构建器
type ReelBuilder struct {
    config *ReelConfig
    rng    *rand.Rand
}

// ReelConfig 轮轴配置
type ReelConfig struct {
    SymbolWeights        []int // 符号权重
    PermutationWeights   []int // 排列权重
    ScatterProb          int   // SCATTER 替换概率（万分比）
    WildProb             int   // WILD 替换概率（万分比）
    TargetLength         int   // 目标长度（默认100）
    AllowWild            bool  // 是否允许 WILD 替换
}

// Build 构建单条轮轴
func (b *ReelBuilder) Build() []int {
    reel := make([]int, 0, b.config.TargetLength)
    lastSymbol := -1

    // 步骤 1+2：生成基础符号序列
    for len(reel) < b.config.TargetLength {
        // 选择符号（排除上一个）
        symbol := b.selectSymbol(lastSymbol)
        lastSymbol = symbol

        // 选择排列
        permLen := b.selectPermutation()

        // 添加到轮轴
        for i := 0; i < permLen && len(reel) < b.config.TargetLength; i++ {
            reel = append(reel, symbol)
        }
    }

    // 步骤 3：特殊符号替换
    b.applySpecialSymbols(reel)

    return reel
}

// selectSymbol 根据权重选择符号（排除指定符号）
func (b *ReelBuilder) selectSymbol(exclude int) int {
    // 计算有效权重总和
    totalWeight := 0
    for i, w := range b.config.SymbolWeights {
        if i != exclude {
            totalWeight += w
        }
    }

    // 随机选择
    r := b.rng.Intn(totalWeight)
    cumulative := 0

    for i, w := range b.config.SymbolWeights {
        if i == exclude {
            continue
        }
        cumulative += w
        if r < cumulative {
            return i + 1 // 符号 ID 从 1 开始
        }
    }

    return len(b.config.SymbolWeights)
}

// selectPermutation 根据权重选择排列长度
func (b *ReelBuilder) selectPermutation() int {
    totalWeight := 0
    for _, w := range b.config.PermutationWeights {
        totalWeight += w
    }

    r := b.rng.Intn(totalWeight)
    cumulative := 0

    for i, w := range b.config.PermutationWeights {
        cumulative += w
        if r < cumulative {
            return i + 1 // 返回 1, 2, 或 3
        }
    }

    return 1
}

// applySpecialSymbols 应用特殊符号替换
func (b *ReelBuilder) applySpecialSymbols(reel []int) {
    for i := 0; i < len(reel); i++ {
        // SCATTER 替换（优先级高）
        if b.rng.Intn(10000) < b.config.ScatterProb {
            reel[i] = SymbolScatter // 9
            continue
        }

        // WILD 替换（仅允许的位置）
        if b.config.AllowWild && b.rng.Intn(10000) < b.config.WildProb {
            reel[i] = SymbolWild // 8
        }
    }
}

// BuildAllReels 构建 5 条轮轴
func BuildBaseReels() [][]int {
    builder := &ReelBuilder{
        config: &ReelConfig{
            SymbolWeights:      []int{1900, 1900, 1500, 1500, 1225, 1050, 925},
            PermutationWeights: []int{8100, 1700, 200},
            ScatterProb:        183,
            WildProb:           275,
            TargetLength:       100,
        },
        rng: rand.New(rand.NewSource(time.Now().UnixNano())),
    }

    reels := make([][]int, 5)

    for col := 0; col < 5; col++ {
        // 第 2、3、4 列（索引 1、2、3）允许 WILD
        builder.config.AllowWild = (col >= 1 && col <= 3)
        reels[col] = builder.Build()
    }

    return reels
}
```

### 5.1 完整实现

```go
// ReelManager 轮轴管理器
type ReelManager struct {
    mu sync.RWMutex

    // 基础模式
    baseReels     [][]int
    baseSpinCount int
    baseInterval  int // 生成间隔

    // 免费模式
    freeReels [][]int
    inFreeGame bool

    // 配置
    baseConfig  *ReelConfig
    freeConfig  *ReelConfig
}

// NewReelManager 创建轮轴管理器
func NewReelManager() *ReelManager {
    return &ReelManager{
        baseInterval: 10,
        baseConfig: &ReelConfig{
            SymbolWeights:      []int{1900, 1900, 1500, 1500, 1225, 1050, 925},
            PermutationWeights: []int{8100, 1700, 200},
            ScatterProb:        183,
            WildProb:           275,
            TargetLength:       100,
        },
        freeConfig: &ReelConfig{
            SymbolWeights:      []int{1600, 1600, 1500, 1500, 1350, 1250, 1200},
            PermutationWeights: []int{8100, 1700, 200},
            ScatterProb:        200,
            WildProb:           590,
            TargetLength:       100,
        },
    }
}

// GetReels 获取当前应使用的轮轴
func (m *ReelManager) GetReels() [][]int {
    m.mu.RLock()
    defer m.mu.RUnlock()

    if m.inFreeGame {
        return m.freeReels
    }

    // 检查是否需要重新生成基础模式轮轴
    if m.baseReels == nil || m.baseSpinCount >= m.baseInterval {
        m.mu.RUnlock()
        m.mu.Lock()
        m.generateBaseReels()
        m.mu.Unlock()
        m.mu.RLock()
    }

    return m.baseReels
}

// generateBaseReels 生成基础模式轮轴
func (m *ReelManager) generateBaseReels() {
    builder := &ReelBuilder{config: m.baseConfig}
    m.baseReels = BuildAllReels(builder)
    m.baseSpinCount = 0
}

// EnterFreeGame 进入免费游戏
func (m *ReelManager) EnterFreeGame() {
    m.mu.Lock()
    defer m.mu.Unlock()

    m.inFreeGame = true
    builder := &ReelBuilder{config: m.freeConfig}
    m.freeReels = BuildAllReels(builder)
}

// ExitFreeGame 退出免费游戏
func (m *ReelManager) ExitFreeGame() {
    m.mu.Lock()
    defer m.mu.Unlock()

    m.inFreeGame = false
    m.freeReels = nil
}

// OnSpinComplete Spin 完成回调
func (m *ReelManager) OnSpinComplete() {
    m.mu.Lock()
    defer m.mu.Unlock()

    if !m.inFreeGame {
        m.baseSpinCount++
    }
}
```

```go
// ValidateReel 验证轮轴生成结果
func ValidateReel(reel []int, config *ReelConfig) error {
    // 1. 检查长度
    if len(reel) != config.TargetLength {
        return fmt.Errorf("长度错误: 期望 %d, 实际 %d", config.TargetLength, len(reel))
    }

    // 2. 统计符号分布
    counts := make(map[int]int)
    for _, sym := range reel {
        counts[sym]++
    }

    fmt.Println("符号分布:")
    for sym := 1; sym <= 9; sym++ {
        fmt.Printf("  符号 %d: %d 个 (%.2f%%)\n", sym, counts[sym], float64(counts[sym])/float64(len(reel))*100)
    }

    // 3. 检查连续相同符号
    maxConsecutive := 1
    currentConsecutive := 1
    for i := 1; i < len(reel); i++ {
        if reel[i] == reel[i-1] && reel[i] != 8 && reel[i] != 9 {
            currentConsecutive++
            if currentConsecutive > maxConsecutive {
                maxConsecutive = currentConsecutive
            }
        } else {
            currentConsecutive = 1
        }
    }

    if maxConsecutive > 3 {
        return fmt.Errorf("发现超过3连的相同符号: %d 连", maxConsecutive)
    }

    fmt.Printf("最大连续相同符号: %d 连\n", maxConsecutive)

    return nil
}
```

## 八、每次Spin取符号机制

### 8.1 核心规则

**关键理解：**
- 轮轴数据：10次spin使用同一套轮轴数据
- 起始位置：**每次spin独立随机新的起始位置**

```
┌─────────────────────────────────────────────────────────────────────┐
│                 每次Spin取符号流程                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  10次Spin使用同一套轮轴数据：                                        │
│  轮轴 = [1, 2, 5, 6, 7, 3, 4, 1, 6, 2, 5, 9, 7, 8, 3, 4, ...]      │
│                                                                      │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  Spin 1: 随机起始位置 = [23, 45, 67, 12, 88]                        │
│  盘面：从各轮轴位置 23,45,67,12,88 各取3个符号                      │
│                                                                      │
│  Spin 2: 随机起始位置 = [56, 78, 34, 91, 15]  ← 新的随机值！        │
│  盘面：从各轮轴位置 56,78,34,91,15 各取3个符号                      │
│                                                                      │
│  Spin 3: 随机起始位置 = [11, 22, 33, 44, 55]  ← 新的随机值！        │
│  盘面：从各轮轴位置 11,22,33,44,55 各取3个符号                      │
│                                                                      │
│  ... Spin 4-10 同样方式，每次起始位置重新随机 ...                   │
│                                                                      │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  Spin 11: 重新生成轮轴数据！                                         │
│  新轮轴 = [新的100个符号序列]                                        │
│  随机起始位置 = [新的随机值]                                         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 8.2 代码实现

```go
// GetGridFromReels 从轮轴生成3x5盘面
func GetGridFromReels(reels [][]int) [3][5]int {
    grid := [3][5]int{}

    for col := 0; col < 5; col++ {
        // 每次Spin都重新随机起始位置！
        start := rand.Intn(len(reels[col]))

        for row := 0; row < 3; row++ {
            pos := (start + row) % len(reels[col])  // 环形取模
            grid[row][col] = reels[col][pos]
        }
    }

    return grid
}
```

### 8.3 Cascading消除后的起始位置变化

```
Cascading消除时，起始位置向前移动：

┌───┐
│ A │ ← 位置 45
│ B │ ← 位置 46

┌───┐

│ ↓ │ ← C下落
│ X │ ← 新符号（从位置 44 取）
└───┘

func (s *SpinState) OnCascade(col int) {
    s.startPositions[col]--
    if s.startPositions[col] < 0 {
        s.startPositions[col] = len(s.reels[col]) - 1  // 环形
    }
}
```
